Para rodar o sistema devemos:

1. Iniciar o servidor: 
    - Dentro da pasta ./backend, digite 'npm start' (Faz nada por enquanto)

2 - Iniciar o cliente:
    - Dentro da pasta ./frontend, 'digite npm start'

3 - Acessar http://localhost:8080/ pelo seu navegador favorito, e desfrutar do nosso petshop ;)
